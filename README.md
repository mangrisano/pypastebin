# pypastebin


What it does
------
PyPastebin is a tool that allows you to generate a pastebin url inside a shell.

Requirements
-----
* A shell :)
* Python3.x

Libraries Used
-----
* Requests

Usage
-----
```
$ cat pypastebin.py | python3 pypastebin.py

Output:
The url is: http://dpaste.com/xxxxxx
```
